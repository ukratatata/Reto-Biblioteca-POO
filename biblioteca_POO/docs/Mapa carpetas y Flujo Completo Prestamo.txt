Ejemplo del Flujo Completo: "Hacer un Préstamo":
Un Empleado va a prestarle un libro a un Socio:

UI: En empleado_view.py, el empleado rellena el ID del socio y el ID del material y hace clic en "Prestar".

UI → Controlador: La vista llama a self._ctrl.realizar_prestamo(id_socio, id_material).

Controlador → Base de Datos: El controlador pide a db.py: "Búscame a este Socio y a este Material".

Base de Datos: Hace un SELECT en SQLite, instancia el objeto Socio y el objeto Libro, y se los devuelve al controlador.

Controlador → Modelos: El controlador usa el modelo: socio.puede_prestar(). Si es falso, aborta y devuelve un mensaje de error a la UI.

Controlador → Base de Datos: Si todo es correcto, el controlador crea un objeto Prestamo nuevo y le dice a db.py: "Guarda todos estos cambios en SQLite".

Controlador → UI: Devuelve un True y un texto de "Éxito".

UI: Muestra la etiqueta verde en pantalla y refresca la tabla visual de préstamos.
